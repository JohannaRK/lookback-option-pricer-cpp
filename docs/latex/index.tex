 \hypertarget{index_autotoc_md1}{}\doxysection{Overview}\label{index_autotoc_md1}
This project implements a European {\bfseries{floating-\/strike lookback option}} pricer under the {\bfseries{Black–\+Scholes}} model using {\bfseries{Monte Carlo}} simulation in {\bfseries{C++}}, with an {\bfseries{Excel/\+VBA}} interface for inputs and visualization.

The implementation outputs\+:
\begin{DoxyItemize}
\item option price
\item \mbox{\hyperlink{structGreeks}{Greeks}}\+: Delta, Gamma, Vega, Rho, Theta
\item charts\+: Price(\+S0) and Delta(\+S0)
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{Key Features}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Black–\+Scholes dynamics with Monte Carlo simulation (C++)
\item Control variate technique to reduce variance (vanilla option benchmark)
\item Open\+MP parallelization ({\ttfamily -\/fopenmp})
\item Price uncertainty output (standard error + confidence interval on price)
\item Two entry points\+:
\begin{DoxyItemize}
\item CLI executable producing CSV outputs
\item Windows DLL exporting VBA-\/friendly functions for Excel
\end{DoxyItemize}
\item Excel dashboard\+:
\begin{DoxyItemize}
\item Run / Reset / Export CSV buttons
\item Price(\+S0) and Delta(\+S0) charts
\end{DoxyItemize}
\end{DoxyItemize}

Excel macros / buttons\+:
\begin{DoxyItemize}
\item Run → {\ttfamily Run\+Dashboard}

\item Export CSV → {\ttfamily Lookback\+\_\+pricer.\+xlsm!\+Export\+CSV}
